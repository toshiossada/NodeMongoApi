"# NodeJs + Express + MongoDB" 
An API Restfull example using MongoDB, NodeJS and Express

Execute 'npm install' after execute 'npm start' to execute project

    
GET     /bills/
    Get list of bills
GET     /bills/:id
    Get bill by id
POST    /bills/
    Insert a new bill
PUT     /bills/:id
    Update a bill
DELETE  /bills/:id
    Delete a bill

GET     /categories
    Get list of categories
GET     /categories/:id
    Get category by id
POST    /categories
    Insert a new category
PUT     /categories/:id
    Update category



GET     /address/:cep
    Get information about address by CEP
